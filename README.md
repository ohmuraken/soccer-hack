# Soccer Hack
A library for dealing with the soccer data easily.

## Data Source
- [Football-Data](http://www.football-data.co.uk/data.php)

## Requirements
### Python
- Python 3.5+

## Installation
```$ pip install git+https://github.com/ohmuraken/soccer-hack.git```
